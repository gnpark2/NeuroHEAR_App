# 📱 NueroHearApp lib code
---
## 📌 개요
이 프로젝트는 난청을 조기에 발견하고, 사용자 스스로 청력 상태를 관리하며,
의사소통 능력을 향상시킬 수 있도록 돕는 것을 목표로 합니다.

단순한 진단을 넘어, 지속적인 훈련과 피드백을 통해 삶의 질 개선을 지향합니다.

---
## ✨ 기능
### 1. 🧪 난청 조기 진단
- 간단한 청력 테스트 제공
- 주파수 기반 청력 반응 측정
- 결과에 따른 난청 위험도 분석

### 2. 📊 청력 상태 관리
- 테스트 결과 저장 및 이력 확인
- 청력 변화 추적 (히스토리 기반)
- 사용자 맞춤 피드백 제공

### 3. 🗣 의사소통 훈련
- 발음 및 청취 훈련 기능
- 음성 인식 기반 피드백
- 상황별 대화 시뮬레이션

### 4. 🤖 개인화 기능
- 사용자 데이터 기반 맞춤 훈련 추천
- 반복 학습 및 개선 유도

---
## 🛠 Tech Stack
### Frontend
- Flutter (FlutterFlow 기반)

### Backend
- Firebase
- Authentication (전화번호 인증)
- Firestore (데이터 저장)
- App Check (보안)

### 기타
- 음성 처리 및 분석 로직
- Web 지원 (Flutter Web)
